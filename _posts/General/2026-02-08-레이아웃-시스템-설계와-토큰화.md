레이아웃 토큰과 디자인 시스템

레이아웃 토큰은 간격, 패딩, 열 수, 그리드 규칙 등을 CSS 변수나 디자인 토큰으로 추상화해 재사용 가능한 구성 요소를 만들고, 서로 다른 뷰에서도 일관된 레이아웃을 구현하는 기법입니다.

name: Build Docs
on:
  push:
    branches: [ main ]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: '18'
      - name: Generate docs
        run: |
          echo "Generating docs..."
          mkdir -p docs
          echo "# LayoutNemo Docs" > docs/index.md
          
      - name: Commit docs
        run: |
          git config user.name "github-actions"
          git config user.email "github-actions@users.noreply.github.com"
          git add docs && git commit -m "docs: generate docs" || true
